# Color Schema

Sample: Dining & Reservation Showcase
Slug: `dining-reservation`
Mood: refined, intimate, evening restaurant atmosphere

## Palette
- Primary: `#2D1E1B`
- Secondary: `#7A3E2B`
- Accent: `#C89B3C`
- Background: `#F8F4EF`
- Surface: `#FFFDFC`
- Text: `#201816`
- Muted Text: `#756861`

## Usage
- Use `#2D1E1B` for hero sections, menu navigation, and footer areas.
- Use `#C89B3C` for reservation buttons, pricing highlights, and active states.
- Support imagery with soft neutrals so food photography remains dominant.
