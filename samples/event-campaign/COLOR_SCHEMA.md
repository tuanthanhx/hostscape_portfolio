# Color Schema

Sample: Event Campaign Website
Slug: `event-campaign`
Mood: festive, urgent, energetic, conversion-led

## Palette
- Primary: `#4C2A85`
- Secondary: `#8B5CF6`
- Accent: `#FFB703`
- Background: `#F8F6FF`
- Surface: `#FFFFFF`
- Text: `#24163D`
- Muted Text: `#6F6590`

## Usage
- Use `#4C2A85` for hero sections, timeline blocks, and registration panels.
- Use `#FFB703` for countdown highlights, tickets, and primary CTA buttons.
- Keep the visual rhythm bold enough for event promotion without hurting readability.
