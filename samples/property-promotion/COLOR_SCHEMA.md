# Color Schema

Sample: Property Promotion Website
Slug: `property-promotion`
Mood: architectural, trustworthy, sales-focused

## Palette
- Primary: `#2F4858`
- Secondary: `#8FA3AD`
- Accent: `#D98E04`
- Background: `#F5F5F2`
- Surface: `#FFFFFF`

- Muted Text: `#68737A`

## Usage
- Use `#2F4858` for map sections, floor-plan labels, and structural UI elements.
- Use `#D98E04` for inquiry buttons and lead-generation highlights.
- Maintain a clean grid that supports plans, specs, and location details.
