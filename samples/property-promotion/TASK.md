# TASK.md

Build a standalone sample website for this folder.

## Sample Identity
- Title: Property Promotion Website
- Slug: `property-promotion`
- Type: Real estate or rental showcase website

## Objective
- Create a property promotion page presenting location, layout, and lifestyle value in a trustworthy way.
- The page should support lead generation and inquiry.
- The final website copy should be written in Japanese for prospective Japanese clients and property seekers.

## Inputs
- Read `../../AGENTS.md`
- Read `../../SAMPLE_SPEC.md`
- Read `../../CODING_RULES.md`
- Read `../../REVIEW_CHECKLIST.md`
- Read `./COLOR_SCHEMA.md`

## Output
- Create `index.html`
- Create `css/styles.css`
- Create `js/scripts.js` only if interaction adds clear value

## Audience
- Japanese users comparing rental or real estate options
- Families, couples, or investors reviewing property details
- Mobile users seeking quick access to plans and location information

## Design Direction
- Mood: architectural, clean, trustworthy
- Prioritize structure, specs, and visual confidence
- The site should feel sales-oriented but not aggressive

## Content Direction
- Use realistic Japanese copy
- Avoid placeholder text
- Highlight location, design value, and practical appeal
- Keep the tone clear, informative, and dependable

## Required Sections
- Hero:
  Japanese property headline, short positioning copy, inquiry CTA
- Property Overview:
  Concept, type, and main appeal points
- Plans and Features:
  Layout examples or specifications
- Location:
  Access, nearby facilities, and neighborhood appeal
- Gallery or Highlights:
  Exterior, interior, or amenity presentation
- Inquiry Reassurance:
  Purchase or contact flow, trust markers, FAQs
- Final CTA:
  Inquiry section
- Footer:
  Company-style information and navigation

## Interaction Guidance
- Map-style highlights, tabs, or hover states are acceptable
- Respect `prefers-reduced-motion`
- Clarity should always outweigh visual effects

## Visual Constraints
- Follow `COLOR_SCHEMA.md`
- Use accent color for inquiry and important property highlights
- Maintain a disciplined, grid-based layout

## Success Criteria
- The page looks portfolio-ready
- It reads as a believable Japanese property promotion website
- It works on mobile and desktop
- It can be reviewed using `../../REVIEW_CHECKLIST.md` without obvious gaps
- The Japanese copy feels appropriate for a real property site
