# Color Schema

Sample: Compact Brand Presentation Site
Slug: `compact-brand`
Mood: concise, stylish, launch-ready

## Palette
- Primary: `#22223B`
- Secondary: `#4A4E69`
- Accent: `#F28482`
- Background: `#F8F7FB`
- Surface: `#FFFFFF`
- Text: `#1E1E24`
- Muted Text: `#6C6C7A`

## Usage
- Use `#22223B` for the main hero, compact nav, and bold type moments.
- Use `#F28482` for CTA buttons and key highlights in the one-page flow.
- Keep sections tight and intentional so the page feels fast and focused.
