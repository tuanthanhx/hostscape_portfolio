# TASK.md

Build a standalone sample website for this folder.

## Sample Identity
- Title: Brand Portfolio Presentation
- Slug: `brand-portfolio`
- Type: Multi-page editorial brand showcase website

## Objective
- Create a showcase website presenting brand work, collaborations, or project highlights in an editorial style.
- The page should let visuals lead while keeping the interface polished and easy to navigate.
- The final website copy should be written in Japanese for prospective Japanese clients and viewers.

## Inputs
- Read `../../AGENTS.md`
- Read `../../SAMPLE_SPEC.md`
- Read `../../CODING_RULES.md`
- Read `../../REVIEW_CHECKLIST.md`
- Read `./COLOR_SCHEMA.md`

## Output
- Create a 4-page static website
- Required pages:
  - `index.html` for the editorial landing and featured work overview
  - `projects.html` for portfolio highlights and case summaries
  - `perspective.html` for philosophy, curation, and collaboration approach
  - `contact.html` for inquiry, client context, and contact guidance
- Keep shared assets inside the sample folder
- Create `css/styles.css`
- Create `js/scripts.js` only if interaction adds clear value

## Audience
- Japanese clients reviewing past work and brand quality
- Visitors interested in featured projects and collaboration range
- Users looking for a clean, curated portfolio experience

## Design Direction
- Mood: gallery-led, editorial, premium
- Let the showcased work carry the page
- Keep the UI restrained and typographically confident

## Content Direction
- Use realistic Japanese copy
- Avoid placeholder text
- Present projects or brand highlights with concise context
- Keep the tone sophisticated and clear

## Required Sections
- Hero:
  Japanese showcase headline, short intro, CTA to featured work
- Featured Projects:
  3 or more portfolio highlights
- Brand Perspective:
  Curation, philosophy, or creative point of view
- Collaboration or Services:
  What kinds of work or partnerships are available
- Trust:
  Clients, recognition, or testimonials
- Final CTA:
  Inquiry or consultation section
- Footer:
  Contact-style details and navigation

## Interaction Guidance
- Hover reveals, simple transitions, or anchored browsing are acceptable
- Respect `prefers-reduced-motion`
- Motion should support browsing, not compete with the content

## Visual Constraints
- Follow `COLOR_SCHEMA.md`
- Use accent color sparingly for featured emphasis
- Preserve an editorial gallery feel

## Success Criteria
- The page looks portfolio-ready
- It reads as a believable Japanese brand showcase website
- It works on mobile and desktop
- It can be reviewed using `../../REVIEW_CHECKLIST.md` without obvious gaps
- The Japanese copy feels appropriate for a real showcase site
