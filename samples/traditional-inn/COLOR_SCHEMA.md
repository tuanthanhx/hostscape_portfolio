# Color Schema

Sample: Traditional Inn Storytelling Page
Slug: `traditional-inn`
Mood: quiet, cultural, crafted, heritage-led

## Palette
- Primary: `#6B2E2E`
- Secondary: `#A67C52`
- Accent: `#D8C3A5`
- Background: `#F6F1E8`
- Surface: `#FFFDF8`
- Text: `#2B2118`
- Muted Text: `#74675B`

## Usage
- Use `#6B2E2E` sparingly for titles, dividers, and key cultural accents.
- Use `#A67C52` for icons, subtle rules, and reservation emphasis.
- Let `#F6F1E8` and `#FFFDF8` carry most of the page for a calm editorial feel.
