# Color Schema

Sample: Accommodation Rebrand Website
Slug: `accommodation-rebrand`
Mood: refreshed, polished, contemporary hospitality

## Palette
- Primary: `#2A4D69`
- Secondary: `#8FB6C9`
- Accent: `#E0A458`
- Background: `#F5F7F8`
- Surface: `#FFFFFF`
- Text: `#1C2830`
- Muted Text: `#687982`

## Usage
- Use `#2A4D69` for brand anchors, typography hierarchy, and hero blocks.
- Use `#E0A458` for upgraded brand accents and reservation interactions.
- The interface should feel cleaner and more current than a legacy hotel site.
