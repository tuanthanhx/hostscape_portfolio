# Color Schema

Sample: Spa & Relaxation Experience Page
Slug: `spa-relaxation`
Mood: serene, elegant, tactile, low-stress

## Palette
- Primary: `#7C6A8A`
- Secondary: `#C8B7D8`
- Accent: `#D9B26F`
- Background: `#F8F5FA`
- Surface: `#FFFFFF`
- Text: `#2B2430`
- Muted Text: `#72677A`

## Usage
- Use `#7C6A8A` for section titles, package tabs, and gentle graphic accents.
- Use `#D9B26F` for booking buttons and premium treatment highlights.
- Keep spacing generous and the interface visually quiet.
