# TASK.md

Build a standalone sample website for this folder.

## Sample Identity
- Title: Spa & Relaxation Experience Page
- Slug: `spa-relaxation`
- Type: Premium spa website

## Objective
- Create a serene spa website that presents treatments, atmosphere, and premium packages with clarity.
- The page should feel luxurious and restful while still driving reservation intent.
- The final website copy should be written in Japanese for prospective Japanese clients and guests.

## Inputs
- Read `../../AGENTS.md`
- Read `../../SAMPLE_SPEC.md`
- Read `../../CODING_RULES.md`
- Read `../../REVIEW_CHECKLIST.md`
- Read `./COLOR_SCHEMA.md`

## Output
- Create `index.html`
- Create `css/styles.css`
- Create `js/scripts.js` only if interaction adds clear value

## Audience
- Japanese users seeking spa, esthetic, or relaxation experiences
- Guests planning premium self-care or gift use
- Mobile users comparing treatment menus and prices

## Design Direction
- Mood: serene, elegant, tactile
- Favor visual quietness, refined spacing, and calm structure
- Avoid loud beauty-industry clichés

## Content Direction
- Use realistic Japanese copy
- Avoid placeholder text
- Present treatments, environment, and care quality in a reassuring way
- Keep the tone soft and premium

## Required Sections
- Hero:
  Japanese spa headline, short promise, reservation CTA
- Spa Concept:
  Brand philosophy and relaxation value
- Treatment Menu:
  3 featured courses or treatments
- Space and Hospitality:
  Rooms, amenities, and service details
- Packages:
  Couple plans, seasonal plans, or premium options
- Guest Trust:
  Testimonials, policy reassurance, or service quality points
- Final CTA:
  Reservation section
- Footer:
  Access, hours, contact-style details, and navigation

## Interaction Guidance
- Gentle reveals and soft hover transitions are acceptable
- Respect `prefers-reduced-motion`
- Keep the interaction layer minimal and calming

## Visual Constraints
- Follow `COLOR_SCHEMA.md`
- Use accent color for booking and premium plan emphasis
- Keep the page airy and quiet

## Success Criteria
- The page looks portfolio-ready
- It reads as a believable premium spa website
- It works on mobile and desktop
- It can be reviewed using `../../REVIEW_CHECKLIST.md` without obvious gaps
- The Japanese copy feels appropriate for a real spa site
