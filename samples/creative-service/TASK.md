# TASK.md

Build a standalone sample website for this folder.

## Sample Identity
- Title: Creative Service Website
- Slug: `creative-service`
- Type: Multi-page creative studio or service website

## Objective
- Create a modular creative service website that explains offerings clearly while still feeling design-forward.
- The page should balance visual confidence with trust and inquiry intent.
- The final website copy should be written in Japanese for prospective Japanese clients and decision-makers.

## Inputs
- Read `../../AGENTS.md`
- Read `../../SAMPLE_SPEC.md`
- Read `../../CODING_RULES.md`
- Read `../../REVIEW_CHECKLIST.md`
- Read `./COLOR_SCHEMA.md`

## Output
- Create a 4-page static website
- Required pages:
  - `index.html` for positioning, selected work, and inquiry entry
  - `services.html` for offer structure and process overview
  - `work.html` for cases, outcomes, and portfolio highlights
  - `contact.html` for consultation flow, FAQ, and inquiry details
- Keep shared assets inside the sample folder
- Create `css/styles.css`
- Create `js/scripts.js` only if interaction adds clear value

## Audience
- Japanese business owners or marketing teams seeking creative support
- Users comparing studios, freelancers, or agencies
- Visitors needing both portfolio confidence and service clarity

## Design Direction
- Mood: sharp, modular, professional
- The layout should feel flexible enough for future expansion
- Avoid vague “creative agency” clichés

## Content Direction
- Use realistic Japanese copy
- Avoid placeholder text
- Present offerings, strengths, and outcomes in a clear and credible way
- Keep the tone modern, articulate, and business-ready

## Required Sections
- Hero:
  Japanese service headline, short positioning copy, inquiry CTA
- Services:
  3 or more service categories with summaries
- Process:
  How the studio works from consultation to delivery
- Portfolio or Cases:
  Highlight example work or results
- Trust Factors:
  Experience, sectors, testimonials, or workflow strengths
- Inquiry CTA:
  Clear contact or consultation section
- Footer:
  Studio details and navigation

## Interaction Guidance
- Subtle motion or content filtering is acceptable
- Respect `prefers-reduced-motion`
- The site should feel deliberate rather than flashy

## Visual Constraints
- Follow `COLOR_SCHEMA.md`
- Use accent color for interaction and inquiry emphasis
- Preserve a strong content hierarchy

## Success Criteria
- The page looks portfolio-ready
- It reads as a believable Japanese creative service website
- It works on mobile and desktop
- It can be reviewed using `../../REVIEW_CHECKLIST.md` without obvious gaps
- The Japanese copy feels appropriate for a real service studio site
