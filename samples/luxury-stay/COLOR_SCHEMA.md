# Color Schema

Sample: Luxury Stay Promotion Site
Slug: `luxury-stay`
Mood: premium, cinematic, warm evening hospitality

## Palette
- Primary: `#1F2A44`
- Secondary: `#7A5C3E`
- Accent: `#D9A441`
- Background: `#F5F1EA`
- Surface: `#FFF9F2`
- Text: `#1A1A1A`
- Muted Text: `#6E675F`

## Usage
- Use `#1F2A44` for hero overlays, main headings, and footer blocks.
- Use `#D9A441` for CTA buttons, booking highlights, and hover states.
- Keep large content areas on `#F5F1EA` and cards on `#FFF9F2`.
