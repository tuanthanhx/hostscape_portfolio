# Color Schema

Sample: Wellness Experience Landing Page
Slug: `wellness-experience`
Mood: soft, restorative, natural, breathable

## Palette
- Primary: `#6F8A64`
- Secondary: `#A7B99A`
- Accent: `#E2B66B`
- Background: `#F7F6F1`
- Surface: `#FEFCF8`
- Text: `#243126`
- Muted Text: `#6D786D`

## Usage
- Use `#6F8A64` for headings, section separators, and ingredient categories.
- Use `#E2B66B` for buttons, plan badges, and key conversion moments.
- Avoid harsh contrast; the page should feel soft and unhurried.
