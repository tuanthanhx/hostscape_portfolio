# Color Schema

Sample: Hotel Information Architecture Build
Slug: `hotel-information`
Mood: dependable, organized, premium-neutral

## Palette
- Primary: `#23405A`
- Secondary: `#7D95A8`
- Accent: `#C9924E`
- Background: `#F2F4F6`
- Surface: `#FFFFFF`
- Text: `#18232D`
- Muted Text: `#63707C`

## Usage
- Use `#23405A` for persistent navigation, page titles, and information blocks.
- Use `#C9924E` for booking prompts and wayfinding highlights.
- Prioritize clarity over atmosphere; this sample should feel scalable and easy to browse.
