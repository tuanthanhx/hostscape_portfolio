# Color Schema

Sample: Resort Landing Experience
Slug: `resort-landing`
Mood: tropical, airy, upscale escape

## Palette
- Primary: `#0F766E`
- Secondary: `#5FB3A2`
- Accent: `#F4B860`
- Background: `#F3FBF8`
- Surface: `#FFFFFF`
- Text: `#17312E`
- Muted Text: `#5B7570`

## Usage
- Use `#0F766E` for navigation, section titles, and image overlays.
- Use `#F4B860` for booking buttons and amenity callouts.
- Keep the layout bright with `#F3FBF8` backgrounds and clean white cards.
